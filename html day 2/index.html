Difference between html tags and elements.

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

Syntax: < tag > < /tag >

Whereas,An HTML element is defined by the combination of start tag, some content, and an end tag.

Syntax: < tag > Content < /tag >

Explain br tag.

The br tag inserts a single line break. The br tag is useful for writing addresses or poems. The br tag is an empty tag which means that it has no end tag.

Explain how many types of lists are there in HTML.

There are 3 types of lists in HTML, namely: Unordered List. Ordered List. Description List.

Unordered lists:

An unordered list typically is a bulleted list of items. HTML gives the ability to customise the bullets.

For example:
  • Item 1
  • Item 2
  • item 3
  • Item 4

Ordered lists:

An ordered list typically is a numbered list of items. HTML gives you the ability to control the sequence number.

For example:
  1. Item 1
  2. Item 2
  3. Item 3

Description lists:

A description list is a list of terms, with a description of each term.

For example:
Item 1
Description
Item 2
Description
Item 3
Description

What are semantic tags?

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.

For example:
  • Header tag
  • Footer tag
  • Main tag,etc.
