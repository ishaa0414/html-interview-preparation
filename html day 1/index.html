Difference between div and span

Div is a block level element and span is an inline element.

Div is always starts on the new line and it will take the whole width of the screen whereas span takes the space required for it,it does not occupy uneccesary width.

For example:

The given text is done using div:

Hi, Good Evening


How are you?

The given text is done using span:

How are you guys?Hope you are doing good.

Difference between class and ID's

ID's and class are selectors and are used to give name to element.Using class and ID's we can target a particular or multiple element.

ID's are unique and classes are not.We can apply multiple classes to any element but we cannot apply the same value of any ID to different element.In one file there should be just one Id of the same name.We denote classes with dot(.) and Id's with hash(#).

What are attributes?

Attributes gives additional information about the tag.It should be given in the lowercase letters.It should be given in opening tags.Multiple attributes can be used in a tag.

For example:

In image tag "alt","width",>etc. are attributes.
